项目经历

物流控制管理系统

软件架构：SpringBoot+MyBatis+MySQL+Redis+RabbitMQ+xxl-job

项目描述：

​ 该项目是为制造业自动化生产过程中使用AGV搬运物料而设计的物流控制管理系统(MCS)。该系统运用先进的技术和自研的任务排配算法，实现了运输调度的智能化和优化，显著降低了人力成本，同时与WMS系统配合，实现生产物料的进出库数据追踪。系统支持根据实际生产场景定制不同的任务模板，从而提高了AGV的使用率和任务的达成率。

项目业绩：

  1. 负责系统后台业务的开发，从0到1打通设备点到点物料的搬运，从设备状态监控，任务生成，任务调度，任务流程控制，任务管理等基本链路。主要的功能模块：任务生成模块、任务分配调度模块和任务管理模块。
  2. 通过 Redis 实现锁，防⽌任务调度任务的重复分配；利用 Redis 实现为每个主任务设置存活时间。
  3. 利用 Redis 的发布订阅功能实现了实时数据更新，提升了系统的性能和实时性。
  4. 结合 RabbitMQ 加快了服务的处理速度，并通过其顺序性消息功能，实现了任务模板中子任务的顺序消费，保证了任务的执行顺序和一致性；同时利用 RabbitMQ 的消息确认机制确保消息的可靠性传输。
  5. 利用 xxl-job 实现了分布式定时任务调度，通过定时任务实现监控任务状态并完成主任务状态的更新。

Qizhi-API 接口开放平台

软件架构：SpringBoot+SpringCloudGateway +Dubbo+Nacos +MySQL +MyBatis-Plus +Hutool 工具库

项目描述：

​ 该API 接口开放调用平台旨在为用户提供现有API 接口，方便用户远程调用，不用用户自己去提供接口服务。管理员可以接入并发布接口，可视化各接口调用情况；用户可以开通接口调用权限、浏览接口及在线调试，并通过客户端SDK 轻松调用接口。

项目业绩：

  1. 根据业务流程，项目的后端划分为接口管理、模拟接口、公共模块、客户端SDK、API 网关这5个子项目。
  2. 为防止接口被恶意调用，设计 API 签名认证算法，为用户分配唯一 ak / sk 以鉴权，保障调用的安全性、可溯源性
  3. 为解决开发者调用成本过高的问题（须自己使用HTTP+ 封装签名去调用接口，平均20 行左右代码），基于 SpringBoot Starter开发了客户端SDK， 一行代码即可调用接口，提高开发体验。
  4. 选用 SpringCloudGateway 作为API 网关，实现了路由转发、访问控制、流量染色，并集中处理签名校验、请求参数校验、接口调用统计等业务逻辑，提高安全性的同时、便于系统开发维护。
  5. 为解决多个子系统内代码大量重复的问题，抽象模型层和业务层代码为公共模块，并使用DubboRPC 框架实现子系统间的高性能接口调用，大幅减少重复代码。

仿某度云盘

软件架构：Springboot + mybatis + mysql+redis+ffmpeg

项目描述：

​ 该项目是一个仿百度云盘面向C端用户的网盘项目，包括用户注册，QQ 快捷登录，文件上传，分片上传，断点续传，秒传，文件在线预览，包括文本，图片，视频，音频，excel，word ，pdf 等文件在线预览，文件分享等功能。

项目业绩：

  • 1.负责用户注册，登录，QQ 快捷登录，发送邮箱验证码，找回密码。
  • 2.负责文件分片上传，秒传，新建目录，预览，文件重命名，文件移动，文件分享，删除，下载 ，回收站等功能。
  • 3.通过 redis 缓存实时计算用户上传过程中空间占用情况。
  • 4.多级目录线性展示，通过递归查询，查询目录的所有父级目录。
  • 5.用户上传文件，同一级目录重名文件自动重命名，文件移动，同名文件重命名。
  • 6.文件分片上传，通过文件 MD 5实现文件秒传，文件分片上传后，异步对文件进行合并处理，视频文件，调用 ffmpeg 生成视频缩略图，将文件分片成 ts 文件。
  • 7.用户通过分享链接和分享码预览下载其他人分享的文件，同时也可以将文件保存到自己的网盘。
  • 8.系统设置模块：超级管理员角色查询所有用户上传的文件，可以对文件下载，删除；超级管理员可以对用户进行管理，给用户分配空间，禁用、启用用户；超级管理员可以对系统进行设置，设置邮箱模板，设置用户注册初始化空间大小。
